Course materials for the Basic VHDL course, found at:
https://vhdlwhiz.com/basic-vhdl-tutorials/

1. Start at exercises/T01_HelloWorldTb.vhd
2. Study the articles and videos and do the exercise yourself
3. Look for TODOs in the code and fill in as needed
4. Compile and simulate in Questa
5. Compare with answers or accompanied blog post

How to install the Questa VHDL simulator for free:
https://vhdlwhiz.com/free-vhdl-simulator-alternatives/
(Questa and ModelSim are interchangeable versions of the same software)

How to create a project, compile, and simulate in Questa,
is shown in the videos for the first few blog posts:
https://vhdlwhiz.com/hello-world/
https://vhdlwhiz.com/wait-for/

Directory structure:

.\answers:
    Folder containing the complete source code for all lessons

.\excercises:
    Folder containing exercises for all lessons
